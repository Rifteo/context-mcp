# Security Policy

## Reporting a Vulnerability

**Do not open a public GitHub issue for security vulnerabilities.**

Report vulnerabilities privately using [GitHub's Security Advisories](../../security).

We will acknowledge within 48 hours and aim to resolve within 14 days.

## Scope

**In scope:**
- Vulnerabilities in the MCP server code or CLI
- Input validation issues that could allow injection via context names or queries
- Dependency vulnerabilities with active exploits

**Out of scope:**
- Vulnerabilities in systems that the loaded contexts describe testing
- Issues with third-party tools referenced in context methodology
- Security issues in your own environment or agent setup
